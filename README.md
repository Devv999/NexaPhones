# 🚀 NexaPhones

**The Future of Intelligent Smartphones**

NexaPhones is a next-generation technology brand focused on building AI-powered smartphones that redefine performance, intelligence, and user experience. By combining advanced hardware with intelligent software, NexaPhones aims to create devices that are not only powerful but truly adaptive.

---

## 🌐 Project Overview

This project represents the official NexaPhones website, featuring:

- Multi-page responsive design  
- Premium UI/UX with smooth animations  
- Product showcase (Nexa One, S, Pro, Ultra)  
- User authentication (Email + Google login)  
- User dashboard (Profile, Wishlist, Orders)  
- Admin panel for product management  
- Early access subscription system  
- Donation/monetization feature  

---

## 🛠️ Tech Stack

### Frontend
- HTML5

- Node.js  
- Express.js  

### Database
